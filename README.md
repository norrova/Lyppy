# Lyppy

A simple, fast and efficient library python to create some project

## How to install
```bash
git clone https://github.com/norrova/Lyppy
cd Lyppy
# Windows...
install.bat
# Linux | MacOS
sh install.sh
```

## Commands
* `lyppy`
    * `-h` : help
    * `-v` : lyppy version
    * `-m` : make
        * `-p` : make project
        * `-mo` : make modules
